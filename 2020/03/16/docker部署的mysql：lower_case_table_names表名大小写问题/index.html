    • docker部署的mysql：lower_case_table_names表名大小写问题

    ﻿#### 问题：

    docker部署的MySQL在进行操作时，出现表名大小写的问题，在MySQL中

    发现
    lower_case_table_names = 0，因此需要将mysql的lower_case_table_names改为1即可

    docker下怎么修改mysql的配置文件

    在配置文件中添加lower_case_table_names=1
