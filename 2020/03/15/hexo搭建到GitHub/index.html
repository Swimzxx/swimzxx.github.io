    二，配置Github

    1,打开同性交友网站 [https://github.com](https://github.com/) 先创建一个仓库

    2,填写仓库的相关信息，点击create repository按钮完成， 填写的name必须为 XXX.github.io

    ​ 创建完成后，本地Git仓库和GitHub仓库之间的传输是通过SSH加密传输的，GitHub需要你的公钥来识别是否是你的推送，所以需要配置ssh key

    3，配置ssh key

    ​ 1>桌面右击，打开Git Bash

    ​ 2>在打开的面板里面输入

    ​ 3> 选择你要将密钥保存的位置，直接回车保存默认位置

    ​ 4>为ssh设置一个密码，直接回车选怎免密

    ​ 5>密钥生成成功，打开C:\Users\Administrator\.ssh可以发现id_rsa和id_rsa.pub两个文件，这两个就是sshkey的密钥对，id_rsa是私钥，id_rsa.pub就是公钥

    6> 打开GitHub的个人设置，配置ssh key title栏随意写，在key文本框中填上粘贴 id_rsa.pub文件里的全部内容。

    7>测试是否成功，在git中输入命令

    8>设置username和email，github每次commit都会记录他们
